# Adverbial clause of condition

- 条件状语从句：
  - 与现在事实相反的条件或结果
    - If + 主语 + 动词的过去式， 主语 + should/would/could/might + 动词原形
  - 与过去事实相反的条件或结果
    - If + 主语 + had + 过去分词， 主语 + should/would/could/might + have + 过去分词
  - 与将来事实可能相反的条件或结果
    - If + 主语 + 动词原形， 主语 + will + 动词原形
