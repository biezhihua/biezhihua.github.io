## 灵饰

### 80级灵饰

```
【80灵饰】
　　获得：可通过灵饰指南书和元灵晶石打造获得，以及通过其他玩家摊位购买、世界喊话收购、以及藏宝阁购买获得
　　80级灵饰分别是：
　　戒指：芙蓉戒
　　耳饰：明月珰
　　手镯：翡玉镯
　　佩饰：逸云佩
　　灵饰效果：鼠标右键点击佩戴，固定加以下属性
　　戒指：物理伤害或物理防御　　
　　佩饰：增加人物速度
　　耳饰：法术伤害或法术防御
　　手镯：封印命中等级或抵抗封印等级
　　说明：
　　1、鉴定时有几率获得狂暴等级、物理暴击、法术暴击、治疗能力等额外属性
　　2、鉴定时有几率获得超级简易特效，可降低使用等级15级(如100级灵饰有该特效，85级则可以使用)
　　3、可以镶嵌星辉石来提高各项能力
```

## 人物

```
【力量属性点】>>　梦幻工具箱
　　☆不同种族力量的转化都是不同的，提高命中和伤害，同时都能增加0.4灵力(1灵力=1法伤+1法防)，0.1速度
　　仙族：1点力量=0.57伤害，0.4灵力，0.1速度
　　人族：1点力量=0.67伤害，0.4灵力，0.1速度
　　魔族：1点力量=0.77伤害，0.4灵力，0.1速度
　　小精灵提醒：
　　1、人物的伤害能力还受攻击修炼影响
　　2、召唤兽加1点力量的效果，增加属性并非固定，跟召唤兽的资质，成长均有关系

【命中属性】
　　作用：命中可提升物理攻击的命中率，同时提升伤害（3点命中=1点伤害）
　　提升：
　　1、增加力量点数，1点力量约1.7命中
　　2、装备镶嵌宝石：红玛瑙
　　3、使用藤蔓妖花变身卡(+100命中)
```
