### 计算机程序设计语言及编译

- 高级语言  High Level Language
  - 自然语言的简洁形式
  - x = 2
- 汇编语言  Assembly Language
  - 引入助记符 
  - MOV X,2
- 机器语言  Machine Language
  - 可以被计算机直接理解
  - C706 0000 0002

- 高级语言 -> 汇编语言 = 编译 Compiling
- 高级语言 -> 机器语言 = 编译 Compiling
- 汇编语言 -> 机器语言 = 汇编 Assembiling

### 编译器在语言处理系统中的位置

- 源程序
- 预处理器 preprocessor
- 经过预处理的源程序
- 编译器
- 汇编语言程序
- 汇编器 Assembler
- 可重定位的机器代码  Relocatable
- 链接器  Linker、加载器 Loader
- 目标机器代码
